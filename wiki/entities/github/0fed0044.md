## Claims
- Memory Tool Path Validation Race Condition Allows Sandbox Escape (Confidence: 0.98, Source: [[0fed0044]])
- The async local filesystem memory tool in the Anthropic Python SDK validated that model-supplied paths resolved inside the sandboxed memory directory, but then returned the unresolved path for subsequent file operations. (Confidence: 0.95, Source: [[0fed0044]])
- A local attacker able to write to the memory directory could retarget a symlink between validation and use, causing reads or writes to escape the sandbox. (Confidence: 0.97, Source: [[0fed0044]])
- Users on the affected versions are advised to update to the latest version. (Confidence: 0.92, Source: [[0fed0044]])
